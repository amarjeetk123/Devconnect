DevConnect is a full-stack social media application inspired by Twitter, designed specifically for developers to connect, share ideas, and engage with each other.

✨ Features
🔐 Authentication – Secure user login & registration
📝 Create Posts – Share thoughts, updates, or ideas
❤️ Like / Dislike – React to posts in real-time
💬 Comments – Engage in discussions on posts
👥 Follow / Unfollow – Build your developer network
📰 Feed System – View posts from people you follow
🛠️ Tech Stack
Frontend: React / Angular (whichever you used)
Backend: Node.js & Express
Database: MongoDB / SQL
Authentication: JWT-based auth
